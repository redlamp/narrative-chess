# Decision — Project Name: Narrative Chess

**Date:** 2026-05-09
**Status:** Adopted (with trade-offs noted)

## Context

Branding audit run on 2026-05-09 to test whether "Narrative Chess" is the right project name before more brand equity (domains, store listings, marketing) compounds. Audit findings live at [[narrative-chess-prior-art]]. Top alternative contenders surfaced: **Pawnsong** (single-word, ownable, evocative) and **Folio Chess** (descriptive + manuscript hint).

Project setting is contemporary — not medieval, not fantasy. That ruled out the manuscript / scribal / mythic axis (Folio, Vellum, Caissa, Bard) for the default-setting brand. Future settings could revisit those words as sub-brands.

## Choice

Keep **Narrative Chess** as the project name.

## Why

- **Plain meaning lands instantly.** Two common words describe the product. New visitors don't need a tagline to understand what it is.
- **Contemporary setting fits the literal name.** "Narrative" reads modern, not period-coded. Reserves room to expand into other settings later without rebranding the umbrella.
- **Cost-to-rename is currently low.** Production sits on `narrative-chess.vercel.app`, no trademark filed, no shipped store listing, no purchased dot-com. The same audit that surfaced the trade-offs documents a clean exit if M2+ traction makes a stronger mark valuable.
- **Sub-brand pattern is open.** "Narrative Chess: <City>" or "Narrative Chess: <Setting>" works once narrative layers ship, the way "Civilization VI: Rise and Fall" extends a generic umbrella.

## Trade-offs accepted

- **Descriptive name is a weak trademark.** Hard to defend against `[Adjective] Chess` competitors. We aren't filing a mark; this is a hobby + indie-ship range project per current ambition.
- **Inactive same-name GitHub repo exists but assessed as non-blocker.** [bh679/Narrative-Chess](https://github.com/bh679/Narrative-Chess) by Brennan Hatton was archived 2026-03-08 with the same name and same concept (browser chess + chess.js + Stockfish, narrative-driven). 0 stars, 0 forks, classifies as abandonware / very early iteration. Reviewed during the audit and determined not to interfere with our identity. Adds minor SERP noise; doesn't constrain branding.
- **Crowded narrative-chess SERP.** Closest competitors share the framing: Chess Story (iOS), Story Time Chess (board game), Chessaria, Stonemachia, Dark Chess, Chess Chronicles, Checkmate Chronicles. Discovery via search will always co-list with these.
- **Genre is well-occupied at every tier.** Children's tabletop, premium narrative campaigns, mobile story-mode, idle/casual chess narratives — all named already. No empty lane.

## Mitigations

- **Anchor identity early.** Register `narrativechess.com` (or the closest available variant) before any marketing pass to claim canonical domain.
- **Differentiate in every public-facing surface.** Landing page H1 + meta description should make positioning unmistakable: "traditional chess with a contemporary narrative layer," not "narrative-driven chess game" (the bh679 repo's tagline).
- **Reserve Pawnsong for the strong-brand path.** If M2+ traction or commercial intent shifts the equation, Pawnsong is the verified-clean rename target. SERP empty, ownable, hints both halves of the project. Captured here so the option doesn't have to be re-discovered.
- **Lean on sub-brand naming for distinctiveness.** As cities, characters, and story beats land, the per-setting name carries the brand weight ("Narrative Chess: <Setting>") and the umbrella stays generic on purpose.

## Alternatives considered

| Candidate | Verdict | Why dropped |
|---|---|---|
| Pawnsong | Reserved for future brand pivot | Best ownable single-word brand. Held back so contemporary setting can stay literal-named for now. |
| Folio Chess | Off-axis | Manuscript hint conflicts with contemporary setting. Could revive as sub-brand for a literary setting later. |
| Stanza | Off-axis + abstract | Same — poetic / literary tone reads period-coded. Too far from "this is chess." |
| Beat / Frame / Scene / Channel / Pulse / Signal Chess | Not pursued | Contemporary axis surfaced after the candidate sweep but before Narrative Chess was confirmed. Available to revisit if a rename is triggered. |
| Chronicle Chess, Saga Chess, Caissa, Vellum, Marginalia | Taken / saturated | See [[narrative-chess-prior-art]] for direct collisions. |
