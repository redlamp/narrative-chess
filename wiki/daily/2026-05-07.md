## Key calls during execution

- **Wordmark aria-label gotcha**: original `aria-label="Narrative Chess - home"` substring-matched both `getByRole("link", { name: "Narrative Chess" })` AND `getByRole("link", { name: "Home" })`. Spec line 64 expected zero "Home" links. Trimmed to `aria-label="Narrative Chess"`.
- **Tailwind v4 custom color tokens**: `border-rule`, `text-oxblood`, `bg-bg-soft` resolve natively via `@theme inline { --color-X: var(--X) }`. No fallback to `style={{}}` needed.
- **GameClient piece pills**: black-piece pill = `bg-black text-white`, white-piece = `bg-white text-black`. These represent literal chess piece colors, so they stay theme-invariant.
- **R3F + theme**: piece materials read `--piece-light` / `--piece-dark` CSS tokens via MutationObserver on `documentElement.class`. Re-fires on theme toggle.
